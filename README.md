# wsl2mydns

Set WSL2 private IP to MyDNS

## target

- WSL2 + Ubuntu

## history

- 2019/07/03 1.0.0 Publish version

## usage

```.sh
npm -g wsl2mydns
wsl2mydns MyDNS-ID MyDNS-PASS
```

wsl2start.bat

```.bat
@echo off
start /b wsl wsl2mydns MyDNS-ID MyDNS-PASS
```
